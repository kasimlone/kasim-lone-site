# Kasim Lone — personal site

Classroom-themed showcase site. Three design variants live at:

- `/literal` — illustrated classroom scene
- `/ide` — full-page code editor aesthetic
- `/hybrid` — illustrated hero + tile grid

Pick one, then we delete the other two and promote the chosen page to `/`.

## Dev

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Push to GitHub, import into Vercel. No env vars required.
